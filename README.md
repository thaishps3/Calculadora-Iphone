# 📱 Calculadora estilo iPhone

## 📝 Descripción
Proyecto de maquetación de una calculadora inspirada en la interfaz de iOS.
Desarrollado utilizando HTML y CSS, con el objetivo de replicar el diseño visual lo más fielmente posible. Actividad perteneciente al curso de "Programación orientada a objetos y base de datos relacionales" del SOC-CIEF. Impartido por el formador Óscar Burgos. 

## 🎯 Objetivo
Practicar:
- Maquetación de interfaces reales
- Uso de CSS Grid y Flexbox
- Organización y limpieza del código
- Detalles visuales (spacing, alineación, jerarquía)

## 🚀 Tecnologías utilizadas
- HTML5
- CSS3 (Grid & Flexbox)

## ⚙️ Funcionalidad
🔹 Versión actual:
- Diseño completamente estático
- Sin lógica en JavaScript

🔹 Próximos pasos:
- Implementar funcionalidad con JavaScript
- Mejorar interacciones (hover, animaciones)
